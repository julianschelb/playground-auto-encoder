3.1 Dataset Preparation¶

In this section we import the CIFAR-10 dataset which consists of 60000 32x32 colour images in 10 classes. For each class 6000 samples are available. The dataset was originally described in the report "Learning Multiple Layers of Features" by Tiny Images, Alex Krizhevsky in 2009 (Chapter 3).

Load Data¶

In [2]:
# Load Data
(x_train, y_train), (x_test, y_test) = cifar10.load_data()

The then classes are encoded as an integer value:

  • 0 = airplane
  • 1 = automobile
  • 2 = bird
  • 3 = cat
  • 4 = deer
  • 5 = dog
  • 6 = frog
  • 7 = horse
  • 8 = ship
  • 9 = truck

Normalize the Data¶

Each pixel is represented as three integer values between 0 and 255 (Red, Green, Blue).

In [4]:
print("Single Pixel:", x_train[1][1][1])

Single Pixel: [145 153 154]

Dividing each RGB value by the max value (255) to scale them between 0 and 1. Doing this no information is lost.

n = 10
plt.figure(figsize=(20, 4))

for i in range(n):
    
    img = x_train[i]
    
    # display original
    ax = plt.subplot(2, n, i+1)
    plt.imshow(img.reshape(32, 32, 3))
    ax.get_xaxis().set_visible(False)
    ax.get_yaxis().set_visible(False)

    # display color distribution
    ax = plt.subplot(2, n, i +1 + n)
    ax.get_xaxis().set_visible(False)
    ax.get_yaxis().set_visible(False)
    
    #Red
    red_values = [pixel[0] for row in img for pixel in row ]
    red = pd.DataFrame({'color': ["red"] * 32 * 32, 'value': red_values } )
    
    # Green
    green_values = [pixel[1] for row in img for pixel in row ]
    green = pd.DataFrame({'color': ["green"] * 32 * 32, 'value': green_values } )
    
    # Blue
    blue_values = [pixel[2] for row in img for pixel in row ]
    blue = pd.DataFrame({'color': ["blue"] * 32 * 32, 'value': blue_values } )
    
    sns.histplot(pd.concat([red, green, blue]), x="value", hue="color", element="step", 
                 palette = ["red", "green", "blue"], ax = ax, legend= False)

plt.suptitle("Color Distribution of the first ten Images after Normalization", fontsize=16)
